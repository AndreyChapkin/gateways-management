Prerequisites:
- java 17 
- NodeJs (tested version 18.12.0)
- npm is available in console
- java is available in console

How to start:
- Enter the project folder (containing gateways-core and gateways-ui projects)
- Run "run-core.bat" in separate console session
- Run "run-ui.bat" in separate console session

Where is available:
- ui is on http://localhost:3000
- core is on http://localhost:8080/swagger-ui.html

How to shutdown gracefully:
- use ctrl + C in the every console session to shutdown each of the servers (core and ui)
